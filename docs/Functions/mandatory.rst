Mandatory
=========

Set-Subscription
----------------

.. _**Synopsis**-45:

**Synopsis**


Sets default subscription. Necessary if in a tenant with multiple
subscriptions.

.. _permissions-28:

Permissions


.. _****Syntax****-45:

****Syntax****



::

  Set-Subscription -Id [Subscription ID]

.. _**Description**-45:

**Description**


Sets the default subscription

.. _**Examples**-45:

**Examples**



::

  Set-Subscription -Id b049c906-7000-4899-b644-f3eb835f04d0

.. _**Parameters**-45:

**Parameters** 


-Id

Subscription ID

.. _required-modules-47:

**Required Modules**


Azure CLI

.. _**Output**-45:

**Output**


Success message
